C++中正则表达式的使用

隐式转换是指由编译器自动执行的的类型转换。假如表达式需要某种特定类型而运算对象是另一种类型，此时只要规则允许，编译器就会自动地将运算对象转换成所需的类型。

何时发生隐式类型转换

在下面这些情况下，编译器会自动地转化运算对象的类型：

  • 在大多数表达式中，比int型小的整形值首先提升为较大的整数类型。
  • 在条件中，非布尔值转换成布尔类型。
  • 初始化过程中，初始值转换成变量的类型；在赋值语句中，右侧运算对象转换成左侧运算对象的类型。
  • 如果算数运算或关系运算对象有多种类型，需要转换成同一种类型。

运算符对象的转换规则

当运算符两侧的运算对象类型不相同时，那么运算符对象将转换成最宽的类型。例如，如果一个运算对象的类型是long double，那么无论另一个运算对象的类型是什么，都会转换成long double。还有一个更普遍的情况，当表达式中既有浮点类型，也有整数类型时，整数值将转换成相应的浮点类型。例：

有时候，我们并不希望编译器在自动转换过程中使用该构造函数。此时，我们可以用explicit关键字声明该构造函数。

C++中正则表达式的使用

总述

正则是一种规则，它用来匹配（进而捕获、替换）字符串。这种规则需要“模式”、“字符串”这两样东西，“模式”根据正则规则，来处理“字符串”。

这种规则被许多语言支持，C++11以后才支持正则。

C++11支持的正则和其他语言支持的正则有区别，本篇记录的目的不在于讲解正则，也不在于阐述C++11支持的正则详细是什么，而是只记下常见的、够用的正则规则（以供速查，应有不严谨之处）及使用用法。
